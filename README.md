# Flutter App

A Flutter application with Hello World text and a dropdown menu.

## Features

- Displays "Hello World" text
- Dropdown menu with "welcome to your firstapp" as first option
- iOS compatible

## Getting Started

To run this app:

1. Make sure you have Flutter installed
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the app

## Requirements

- Flutter SDK >= 3.0.0
- iOS development environment (for iOS builds)
